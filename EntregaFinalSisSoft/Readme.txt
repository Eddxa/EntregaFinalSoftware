El proyecto de sistema de tickets nace como una herramienta de gestión de incidentes y solicitudes 
que permite a las empresas recibir, asignar, clasificar,  y resolver problemas de manera 
eficiente y ordenada. Con este software, los usuarios pueden generar un ticket con su solicitud, 
requerimiento o problema, el cual se registra en la plataforma y se le asigna automáticamente a un 
agente de soporte específico para su resolución. El sistema deberá contar con una interfaz fácil de 
usar para que los usuarios puedan crear y administrar sus solicitudes, así como para que los agentes 
puedan asignar y dar seguimiento a los tickets. Asimismo, el sistema también proporcionará informes 
estadísticos para ayudar a los gerentes a mejorar la calidad del servicio al cliente y reducir los 
tiempos de resolución de requerimientos.
